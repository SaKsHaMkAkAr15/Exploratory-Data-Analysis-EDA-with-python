# Titanic Exploratory Data Analysis (EDA)

## Main Aim of the Project
Exploratory Data Analysis on the Titanic dataset to find insights about passenger survival rates using Python, Pandas, Matplotlib and Seaborn through jupyter notebook.

## Key Insights Found In This Project
- Only 38.38% of passengers survived
- Female survival rate was 74.2% vs male survival rate of 18.89%
- 1st class survival rate was 62.96% vs 3rd class survival rate of 24.24%
- Most passengers were between 20-25 years old
- Passengers who paid higher fares had better survival chances

## Steps to Run this project
1. Make sure Python is installed
2. Install required libraries:

pip install pandas matplotlib seaborn jupyter

3. Download train.csv from Kaggle.com Titanic dataset
4. Run this command in the terminal:

jupyter notebook

5. Open titanic_eda.ipynb and run all cells
